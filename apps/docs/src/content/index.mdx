# Introduction

Never handle flows manually again. Kickstart your next project with pre-configured authentication, UI components, and best practices baked in.

## Features

- **Multi-Factor Authentication** - Support for SMS, Email OTP, and authenticator apps
- **Social Sign-In** - Pre-built integration with popular providers
- **Account Management** - User profiles, account linking, and settings
- **Email Templates** - Customizable templates for verification and notifications
- **UI Components** - Ready-to-use, accessible components
- **TypeScript Support** - Full type safety and auto-completion
- **Modern Stack** - Next.js, Tailwind CSS, Radix UI
- **Developer Experience** - Fast refresh, strong linting, and formatting
