# Admin Panel

The Harvide Starter admin panel provides a comprehensive interface for managing your application, users, and system settings.

## Configuration

Enable and configure the admin panel in your `starter.config.ts`:

```typescript
export default {
  admin: {
    enabled: true,
    impersonationSessionDuration: 3600, // 1 hour
    defaultBanReason: "Violation of terms of service",
    defaultBanExpiresIn: 7 * 24 * 3600, // 7 days
    bannedUserMessage: "Your account has been suspended",
    dashboard: {
      shortcuts: [
        {
          name: "Users",
          description: "Manage user accounts",
          href: "/admin/users"
        },
        {
          name: "Settings",
          description: "System configuration",
          href: "/admin/settings"
        }
      ],
      metrics: [
        {
          name: "Total Users",
          type: "count",
          query: "SELECT COUNT(*) FROM users"
        },
        {
          name: "Active Users",
          type: "count",
          query: "SELECT COUNT(*) FROM users WHERE status = 'active'"
        }
      ]
    }
  }
}
```

## User Impersonation

### Configuration

```typescript
export default {
  admin: {
    impersonation: {
      enabled: true,
      sessionDuration: 3600,
      requireReason: true,
      logActions: true
    }
  }
}
```

### Usage

1. **Start Impersonation**
   ```typescript
   await admin.impersonate({
     userId: "user_123",
     reason: "Support assistance",
     duration: 3600
   });
   ```

2. **End Impersonation**
   ```typescript
   await admin.endImpersonation({
     userId: "user_123"
   });
   ```

## User Banning

### Configuration

```typescript
export default {
  admin: {
    banning: {
      enabled: true,
      defaultReason: "Terms violation",
      defaultDuration: 7 * 24 * 3600,
      requireReason: true,
      notifyUser: true
    }
  }
}
```

### Usage

1. **Ban User**
   ```typescript
   await admin.banUser({
     userId: "user_123",
     reason: "Spam",
     expiresIn: 24 * 3600
   });
   ```

2. **Unban User**
   ```typescript
   await admin.unbanUser({
     userId: "user_123"
   });
   ```

## Dashboard Customization

### Shortcuts

Configure quick access links:

```typescript
shortcuts: [
  {
    name: "Analytics",
    description: "View system metrics",
    href: "/admin/analytics",
    icon: "ChartBarIcon"
  }
]
```

### Metrics

Add custom metrics:

```typescript
metrics: [
  {
    name: "New Users (24h)",
    type: "count",
    query: `
      SELECT COUNT(*)
      FROM users
      WHERE created_at > NOW() - INTERVAL '24 hours'
    `
  }
]
```
