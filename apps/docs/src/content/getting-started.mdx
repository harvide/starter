# Getting Started

Harvide Starter provides a complete foundation for building modern web applications. Get started quickly with our CLI tool.

## Quick Start

```bash
npx create-harvide-starter
```

The CLI will guide you through setup with the following options:

1. **Project Details**
   - Project name
   - Description
   - Package manager (`npm`, `pnpm`, `yarn`, `bun`)

2. **Features**
   - Web Application (Next.js)
   - Admin Panel
   - Authentication Methods:
     - Email/Password
     - Phone Number
     - Social Providers (Google, Apple, LinkedIn)

3. **Email Setup**
   - Provider selection (Resend/SMTP)
   - From address configuration
   - Template customization

4. **Database**
   - Connection string
   - Initial migration
   - Environment setup

Once configured, start your development server:

```bash
cd your-project-name
npm run dev # or yarn/pnpm/bun
```

Visit `http://localhost:3000` to see your application.
