# Welcome to @harvide/starter

**The foundation for your next application, built for speed and flexibility.**

@harvide/starter is a production-ready boilerplate that provides everything you need to build modern web applications. Focus on your business logic while we handle the complexities of authentication, user management, payments, and more.

@harvide/starter uses Next.js as a foundation, with BetterAuth for authentication and user management, multiple payment providers such as Polar or Stripe and DrizzleORM for database interactions. It comes with a powerful admin panel, email service and easy configuration through a single file.

## What's Included

### Core Features
- **Authentication**: Email/password, social providers, phone auth
- **User Management**: Profiles, roles, permissions
- **Admin Panel**: User management, analytics, settings
- **Email Service**: Transactional emails, templates, multiple providers

### Payments & Monetization
- **Stripe Integration**: Subscriptions, usage-based billing
- **Polar Integration**: Open source monetization
- **Multiple Providers**: Flexible payment options

### Developer Experience
- **Single Config**: Control everything through `starter.config.ts`
- **Type Safety**: Full TypeScript support
- **Modern Stack**: Next.js, Tailwind CSS, Drizzle ORM
- **AI Ready**: Optimized for AI code assistants

## Getting Started

1. Create a new project:
```bash
npx create-harvide-starter
```

2. Configure your application in `starter.config.ts`

3. Start building!
